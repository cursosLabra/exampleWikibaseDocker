# exampleWikibaseDocker

This repository is just an example of a Wikibase docker which is a mirror of the [example folder](https://github.com/wmde/wikibase-release-pipeline/tree/main/example).

## Instructions

A TL;DR instructions can be:

1. Clone this repo
1. Edit or review the `.env` file (it can work as is, but you may prefer to customize it)
1. run `docker-compose -f docker-compose.yml -f docker-compose.extra.yml up`

The full instructions are [here](https://www.mediawiki.org/wiki/Wikibase/Docker)
